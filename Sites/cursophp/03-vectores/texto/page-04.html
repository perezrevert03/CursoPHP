Imprimiendo el precio de los productos

Se quiere modificar el sistema de pedidos de la Tienda de recambios de Bob para que el formulario muestre los precios de los productos a partir del contenido de un vector.

Modificar el fichero Sites/cursophp/03-vectores/ficheros/formulario.php para que contenga una matriz asociativa $precios similar al vector $productos definido con anterioridad, pero siendo el valor de cada elemento un vector con la descripción del producto y el precio.

Ejemplo de un elemento del nuevo vector $precios.

<?php
$precios= array(
   "aceite" => array("desc" => "Lata de aceite",
                     "precio" => 10),
   ...
   );
?>

Editar el fichero formulario.php para que utilizando el vector $precios imprima al lado de cada producto el precio por unidad.

El resultado deberá ser similar a la tabla que se muestra a continuación.

Elementos Cantidad
Frenos (100 €/ud)
Latas de aceite (10 €/ud)
Neumáticos (4 €/ud)
Bujías (2 €/ud)

Comprobar el nuevo formato del formulario ejecutando el fichero formulario.php.
