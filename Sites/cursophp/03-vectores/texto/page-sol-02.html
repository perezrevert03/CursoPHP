Automatizando el formulario de la tienda de Bob

Se pretende generar el formulario de petición de la tienda de Bob de forma automática a partir del contenido de un vector de productos.

Añadir al fichero Sites/cursophp/03-vectores/ficheros/formulario.php un vector asociativo $productos que contenga el nombre de los productos como valores (Frenos, Latas de aceite y Neumáticos) y los identificadores de dichos productos como claves (frenos, aceite y ruedas).

Modificar la tabla con los campos de entrada del formulario para que se genere mediante un bucle. Cada fila de la tabla se generará a partir de uno de los elementos del vector $productos. Como atributo name de cada elemento input se utilizarán las claves del vector producto concatenadas con el prefijo num: "num" . $clave o "num$clave" (v.g. numaceite).

Comprobar que el formulario se imprime correctamente accediendo al fichero formulario.php.

Comprobar la corrección del ejercicio anterior añadiendo al vector de productos una nueva entrada: bujias ⇒ Bujias. Comprobar que se añade una nueva fila al formulario mostrado por formulario.php conteniendo el nuevo campo de entrada.
