Un ejemplo sencillo con matrices

Vamos a trabajar con algunos ejemplos de manipulación de matrices.

Editar el fichero Sites/cursophp/03-vectores/ficheros/matrices.php para definir una matriz de 3 x 3 que contenga los números del 1 al 9. Hay que recordar que en PHP una matriz no es más que un vector de fila.

La matriz deberá tener un aspecto como el que se muestra a continuación:

<?php
$matriz= array(
    array(...),
    array(...),
    array(...)
  );
?>

Añadir al fichero Sites/cursophp/03-vectores/ficheros/matrices.php el código necesario para imprimir la matriz dentro de una tabla HTML. Se sugiere utilizar dos bucles for.

Se debe recordar que para imprimir una matriz serán necesarios dos bucles anidados: uno para obtener los vectores fila y otro que recorra cada vector fila e imprima un valor en cada celda de la tabla.
