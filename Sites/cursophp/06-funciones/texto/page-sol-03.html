Calculando el coste del pedido mediante funciones

Se dese completar la modificación del script peticion.php para que el cálculo del coste del pedido se realice mediante una función.

Editar el módulo Sites/cursophp/06-funciones/ficheros/boblib.php y el fichero Sites/cursophp/06-funciones/ficheros/peticion.php para añadir una función que calcule el coste del pedido. La nueva función coste_pedido(...) deberá recibir la tabla de productos y el vector de pedidos y devolver el coste total sin impuestos.

Comprobad la nueva funcionalidad realizando un pedido desde formulario.php.
