Generación de una factura

Se pretende añadir ahora la funcionalidad necesaria para la generación de una factura del pedido realizado.

Modificar los ficheros peticion.php y boblib.php para que la función coste_pedido() devuelva en un tercer parámetro pasado por referencia un vector con el coste parcial para cada producto solicitado (precio del producto x cantidad).

Se suministra el fichero Sites/cursophp/06-funciones/ficheros/factura.php que genera una tabla similar a una factura.

Completar el fichero Sites/cursophp/06-funciones/ficheros/factura.php convirtiendo la tabla existente en una función imprime_factura(...) que imprima dicha factura a partir de los parámetros: nombre, dirección, el vector de productos, el vector de pedidos y el vector de costes parciales y el total.

Los campos que todavía son desconocidos, como el nombre o la dirección, se deberán dejar en blanco.

Incluir el fichero factura.php desde el script peticion.php e invocar la nueva función imprime_factura() al final del script.
