Obteniendo información del servidor

Vamos a probar como PHP puede acceder a la información suministrada por el servidor web. Para ello se va a utilizar la función phpinfo() y el vector $_SERVER.

Editar el fichero Sites/cursophp/02-lenguaje-php/ficheros/funcion.php para que muestre información sobre el sistema mediante la invocación de la función phpinfo().

Observad la información de la instalación ejecutando ahora el fichero funcion.php.

Vamos a hora a añadir a los mensajes anteriores la información relativa al tipo de navegador utilizado para acceder a la página funcion.php. La información está disponible en el elemento HTTP_USER_AGENT del vector $_SERVER. Se puede ver un ejemplo en el manual de PHP, capítulo Conceptos básicos → Una explicación sencilla → Algo útil.

Editar el fichero Sites/cursophp/02-lenguaje-php/ficheros/funcion.php para que imprima la información del navegador utilizado.

Observad la información del navegador ejecutando el fichero funcion.php.

Esta información se podría utilizar para generar un contenido específico en una página web en función del navegador utilizado para accederla.
