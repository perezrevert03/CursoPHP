Marcas de inserción de código PHP

Se pretende probar el conjunto de marcas de inserción de código PHP que están activas en la configuración del equipo de prácticas.

Editar el fichero Sites/cursophp/02-lenguaje-php/ficheros/prueba.php para que imprima varios mensajes de saludo utilizando la sentencia echo. Cada mensaje de saludo deberá utilizar una de las diferentes marcas de inserción de código permitidas en PHP.

Recordar que los estilos permitidos eran: SGML, XML, SCRIPT y ASP, donde los estilos SGML y ASP permiten una versión reducida de la sentencia echo.

La instalación de PHP realizada en el laboratorio podría tener algunas de las marcas deshabilitadas en su configuración.

Probar su funcionamiento pinchando en el siguiente enlace: prueba.php

Las marcas de inserción de código que no estén habilitadas normalmente quedarán sin procesar y aparecerá el código PHP en el fichero HTML que se le pase al navegador, el cual también las ignorará. Comprobadlo viendo el código HTML generado con ayuda del navegador.

El código fuente se puede obtener pulsando el botón derecho sobre el navegador y escogiendo la opción Ver código fuente de la página.
