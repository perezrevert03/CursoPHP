Introducción

El entorno de trabajo que vamos a utilizar durante las sesiones prácticas está formado por los siguientes componentes:

  • Sistema operativo: GNU/Linux. Distribución Ubuntu (14.04).
  • Entorno gráfico: Gnome.
  • Servidor web: Apache (2.4)
  • Servidor de bases de datos: MySQL (5.5).
  • PHP versión 5.5

Algunos de estos componentes no son indispensables para el desarrollo de páginas web mediante PHP, pero son los que se van a utilizar durante este curso.

El sistema operativo Linux es un sistema multitarea multiusuario, por lo que, antes de comenzar a trabajar, hay que darse de alta en el sistema. El usuario que se ha creado para la realización del curso se denomina alumno y la clave utilizada es cursophp.

Editores

Para editar los ficheros PHP, que son ficheros de texto normales, como los ficheros HTML, se pueden utilizar cualquier editor disponible en el sistema. Para acceder a ellos, o se teclea el comando en una consola del entorno gráfico, como puede ser gedit o geany (recomendado), o se utiliza el menú principal de GNOME Classic: Aplicaciones → Accesorios → Editor de textos, Aplicaciones → Programación → Geany, etc.

En la barra de acceso rápido se han colocado las aplicaciones más habituales, como el navegador de internet (firefox), el explorador de ficheros (nautilus), el editor geany y el terminal con el interprete de ordenes.

Estructura de directorios

Para que los ficheros generados sean accesibles por el servidor web se colocan en un directorio especifico. En la actual instalacion ese directorio es Sites/cursophp, debe encontrarse en el directorio HOME del usuario y tener permisos de lectura para todos los usuarios del sistema (en concreto para el servidor apache).

drwxr-xr-x   15 alumno    alumno        4096 Sep  2 16:50 Sites/cursophp

Para la realización de cada sesión práctica crearemos un directorio que contendrá los ficheros correspondientes a dicha sesión. A este directorio lo denominaremos directorio de trabajo. Los directorios se crearán dentro del directorio Sites/cursophp.

Así pues, para esta sesión práctica crearemos un directorio que se denominará 02-lengueje-php. Utilizar para ello la orden mkdir o el gestor de ficheros de GNOME: Lugares → Carpeta personal.

Navegador web

Para comprobar el correcto funcionamiento de las pruebas realizadas se utilizará uno de los navegadores disponibles: mozilla, netscape, konqueror, etc. Estos navegadores deberían estar disponibles en el menú Aplicaciones → Internet.

Para visualizar los ficheros web de nuestro usuario se utilizará la URL: http://cursophp.local/.

Documentación

Toda la documentación necesaria para la realización del curso estará disponible a lo largo del curso en la dirección http://cursophp.disca.upv.es/. Además de las transparencias, boletines de las sesiones prácticas y algún material adicional, en esta dirección encontraréis disponible importante documentación en línea.

PHP, al igual que otros lenguajes de script, ofrece multitud de funciones para acceder a la amplia gama de servicios a los que permite acceder. Es práctica habitual y recomendable en estos casos, el tener constantemente un navegador apuntando a la documentación en línea que oferce el lenguaje PHP, tanto para consultar dudas, como para buscar ejemplos, funciones nuevas, etc. En la página web del curso están disponibles distintas versiones de dicha documentación. Se recomienda encarecidamente que se utilice la documentación disponible sobre PHP.

Para la visualización de la documentación que se encuentra en PDF se puede utilizar el visor evince o xpdf.

Ficheros de ayuda

Algunas sesiones prácticas dispondrán de ficheros de ayuda (esqueletos o ejemplos) para su realización. Estos ficheros estarán disponibles en la página web del curso. Los ficheros de ayuda estarán enpaquetados con la utilidad tar. Para descomprimirlos se puede utilizar la orden tar xvzf fichero.tgz. Si está orden se ejecuta sobre el directorio ${HOME}/Sites/cursophp la descompresión del fichero generará automáticamente el directorio de trabajo.
